# Mock Interview AI

An AI-powered web platform for practicing technical interviews. It generates tailored questions using the **Groq LLaMA 3** model, listens to your spoken answers via the **Web Speech API**, evaluates them with **NLP**, and provides behavioral feedback from a live webcam stream analyzed with **MediaPipe**.

---

## ✨ Features

| Feature | Details |
|---|---|
| Dynamic question generation | Groq LLaMA 3 generates technical, behavioral & case-study questions |
| Speech-to-text | Browser-native Web Speech API (no extra setup) |
| NLP evaluation | `sentence-transformers` semantic similarity + keyword matching |
| Follow-up questions | Score-adaptive follow-ups generated by the LLM |
| Computer vision | MediaPipe FaceMesh head-pose estimation on live webcam |
| SQLite persistence | All sessions, questions & answers stored via Flask-SQLAlchemy |
| Final report | Detailed per-question breakdown with animated score bars |

## ⚙️ Setup Guide

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd mock-interview-ai
```

### 2. Create and activate a virtual environment

```bash
python3 -m venv venv
source venv/bin/activate          # macOS / Linux
# venv\Scripts\activate           # Windows
```

### 3. Install Python dependencies

```bash
pip install -r requirements.txt
```

> **Note:** `torch` can be large (~2 GB). If you only need CPU inference, install the CPU-only wheel first:
> ```bash
> pip install torch --index-url https://download.pytorch.org/whl/cpu
> ```

### 4. Set the Groq API key

Obtain a free API key from [console.groq.com](https://console.groq.com) and export it:

```bash
export GROQ_API_KEY="gsk_your_key_here"
```

Add it to `~/.zshrc` (or `~/.bashrc`) to make it permanent.

### 5. Run the application

```bash
cd backend
python main.py
```

The server starts on **http://localhost:5000**. Open that URL in **Chrome** or **Edge** (required for the Web Speech API).

## 🚀 Usage

1. **Select a role** from the dropdown (e.g., *Python Developer*).
2. **Enter your skills** (e.g., `Django, AWS, Docker`). Use the quick-add tag buttons.
3. Click **Start Interview** — the LLM generates ~9 tailored questions.
4. Allow **microphone** and **camera** permissions when prompted.
5. Click **Start Answering**, speak your response, then click **Stop Recording**.
6. View your evaluation score and feedback after each answer.
7. After the final question, click **View Full Report** for a detailed breakdown.

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Backend | Python 3.11+, Flask 3, Flask-CORS, Flask-SQLAlchemy |
| LLM | Groq API (`llama3-8b-8192`) |
| NLP | `sentence-transformers` (`all-MiniLM-L6-v2`), `nltk`, `scikit-learn` |
| Computer Vision | `opencv-python`, `mediapipe` |
| Database | SQLite |
| Frontend | Vanilla HTML5, CSS3, JavaScript (no frameworks) |
| Speech | Web Speech API (`webkitSpeechRecognition`) |

## 🔒 Environment Variables

| Variable | Required | Description |
|---|---|---|
| `GROQ_API_KEY` | ✅ Yes | Your Groq API key |
| `FLASK_SECRET_KEY` | No | Custom secret key for Flask sessions (defaults to a dev key) |
| `PORT` | No | Port to run on (default: `5000`) |

## 🐞 Troubleshooting

| Problem | Solution |
|---|---|
| `GROQ_API_KEY is not configured` | Export the env variable before running `python main.py` |
| Speech not working | Use Chrome/Edge; grant microphone permission in browser |
| Webcam feed blank | Grant camera permission; check no other app is using the webcam |
| `mediapipe` install fails | Try `pip install mediapipe==0.10.11` explicitly |
| Slow startup | `sentence-transformers` downloads the model on first run (~90 MB) |
